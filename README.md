#### Python版本

+ 推荐 Python3 (3.6 - 3.9)

#### 已有功能
  - [x] 自动观看课程
  - [x] 课程选择
  - [x] 自动跳过已完成章节
  - [x] 自动答题
      - [x] 已过期题目作答（成绩可能无效，待测）
      - [x] 复制另一个用户的答案自动作答
  - [x] 导出题目与答案（支持已结课的课程）
